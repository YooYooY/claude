---
name: commit-message
description: Generates a commit message based on Git changes. Intended for use when a user needs to provide a commit description or message.
---

1. First, run `git status` and `git diff` in working directory to review the changes.
2. Write a one-line commit title, following the Conventional Commits style; add brief bullet points if necessary.
3. If there are no changes, indicate that there is nothing to commit.
